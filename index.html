Some Projects I've Been Working On

NASA's Mars Rover Photo API

This is a web API for accessing photos from NASA's Mars rovers: Curiosity, Opportunity, and Spirit. It began as a student project while I was attending Launch Academy, a coding bootcamp in Boston. It's now hosted at api.nasa.gov as part of NASA's Open API Portal. Here is its source code on GitHub.

Resque

I'm lucky enough to be on the team of talented people maintaining Resque, an open source backgrounding system written in Ruby.

AsciiArt

This is a silly little side project of mine: an interactive tool for creating ASCII art masterpieces written with Ruby's Curses library. It's published at rubygems.org and the source code is on GitHub.

Goober

Another side project I've been enjoying working on with some friends, Goober is a 2d puzzle game written in Rust using the Piston game development library.
